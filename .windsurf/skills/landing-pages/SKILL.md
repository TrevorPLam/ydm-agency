## Purpose
Guide landing page implementation with conversion optimization, A/B testing, and analytics integration for marketing campaigns.

## When to Invoke
- When implementing landing pages
- When setting up conversion tracking
- When implementing A/B testing

## Prerequisites
- Next.js configured
- Analytics configured

## Implementation Steps

### 1. Design landing page structure
Use the template in `templates/landing-page.tsx`

### 2. Add conversion tracking
Reference `templates/conversion-tracking.tsx`

### 3. Implement A/B testing
Use `templates/ab-testing.tsx`

### 4. Add lead capture forms
Use `templates/lead-form.tsx`

### 5. Configure analytics
Use `templates/analytics-config.ts`

## Templates
- `templates/landing-page.tsx` - Landing page template
- `templates/conversion-tracking.tsx` - Conversion tracking
- `templates/ab-testing.tsx` - A/B testing
- `templates/lead-form.tsx` - Lead capture form
- `templates/analytics-config.ts` - Analytics configuration

## Patterns
- Use above-the-fold CTAs
- Implement social proof
- Track conversion events

## Anti-Patterns
- Don't clutter with distractions
- Don't ignore mobile optimization
- Don't skip A/B testing
