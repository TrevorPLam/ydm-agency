# Remote Cache Monitoring Checklist

## Setup
- [ ] Link project to Vercel
- [ ] Configure team scope
- [ ] Set TURBO_TOKEN environment variable
- [ ] Set TURBO_TEAM environment variable
- [ ] Test remote cache with `turbo run build --remote`

## Monitoring
- [ ] Check cache hit rate in Vercel dashboard
- [ ] Monitor cache size
- [ ] Review cache miss patterns
- [ ] Optimize cache keys if needed

## Maintenance
- [ ] Clear cache if corrupted
- [ ] Update cache configuration as needed
- [ ] Review cache costs monthly
