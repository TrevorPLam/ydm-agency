## Purpose
Guide form implementation with validation, submission, and multi-step forms using React Hook Form patterns.

## When to Invoke
- When implementing form packages
- When setting up form validation
- When creating multi-step forms

## Prerequisites
- React configured
- React Hook Form installed

## Implementation Steps

### 1. Set up form validation schema
Use the template in `templates/validation-schema.ts`

### 2. Implement form components
Reference `templates/form-component.tsx`

### 3. Add multi-step form logic
Use `templates/multi-step-form.tsx`

### 4. Configure form submission
Use `templates/form-submission.ts`

### 5. Add form analytics
Use `templates/form-analytics.tsx`

## Templates
- `templates/validation-schema.ts` - Validation schema
- `templates/form-component.tsx` - Form component
- `templates/multi-step-form.tsx` - Multi-step form
- `templates/form-submission.ts` - Form submission
- `templates/form-analytics.tsx` - Form analytics

## Patterns
- Use Zod for validation
- Implement progressive disclosure
- Track form abandonment

## Anti-Patterns
- Don't ignore validation errors
- Don't skip accessibility
- Don't hardcode form fields
